# Users & Todos – React Assignment

A small React + TypeScript project using the JSONPlaceholder API.  
The app shows a list of users, allows selecting a user to view their TODOs,  
and includes a “Hide completed” filter with URL state persistence.

---

## Run the project

```bash
npm install
npm run dev

---

## Tech Stack
React, TypeScript, React Router, Redux, SCSS.
